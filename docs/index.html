Features

🎤

Push-to-Talk

Microphone is only active while you hold the key. No always-on listening, no background recording.

🌐

Works Everywhere

Types into any focused window — terminals, browsers, editors, chat apps, Word, anything with a text cursor.

🔒

100% Offline

Runs entirely on your machine using Whisper. No cloud APIs, no data leaves your computer. Ever.

⚡

GPU Accelerated

Metal on Apple Silicon, CUDA on NVIDIA GPUs. Transcribes a 3-second clip in about 1 second.

🇧🇦

Bilingual

Auto-detects English and Spanish. Just speak naturally — TAK figures out the language.

🔴

Visual Overlay

A floating pill appears on all screens while recording (red) and transcribing (yellow). Always know the state.

Get Started

Up and running in under 2 minutes.

# Clone and run the installer (macOS or Linux)
git clone https://github.com/lchonkan/tak.git
cd tak
./install.sh

# The installer detects your platform, installs
# dependencies, sets up the conda environment,
# and verifies everything works.

# Then run:
./run.sh
# Install system dependencies
brew install portaudio ffmpeg

# Create environment and install
conda create -n tak python=3.11 -y
conda activate tak
git clone https://github.com/lchonkan/tak.git
cd tak
pip install -r requirements-macos.txt

# Grant Accessibility permission
# System Settings → Privacy & Security → Accessibility
# → Add your terminal app

# Run!
./run.sh
# Install system dependencies
sudo apt install xdotool xclip libportaudio2

# Create environment and install
conda create -n tak python=3.11 -y
conda activate tak
git clone https://github.com/lchonkan/tak.git
cd tak
pip install -r requirements-linux.txt

# For GPU acceleration (recommended)
pip install nvidia-cublas-cu12 nvidia-cudnn-cu12

# Grant input permissions
sudo usermod -aG input $USER
# Log out and back in, then:

# Run!
./run.sh
